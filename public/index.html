Chattanooga Tax Lookup API

Backend service for the Joe Leffew Properties property tax lookup tool. This page is the public root; the actual API lives at the endpoint below.

Endpoint

GET /api/lookup?address=1524+Green+Pond+Rd&price=750000

Parameters

  • address — required. Hamilton County street address.
  • price — optional. Purchase price for tax projection.

This is an internal API. For tax estimates, use the calculator on joeleffew.com.
